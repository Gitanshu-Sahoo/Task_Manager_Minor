# Task_Manager_Minor
Minor Project made by Gitanshu Sahoo, UID: WD-FSWD-A4/May-10208

Description:
Task Manager is a responsive application created using HTML, CSS, and JavaScript to make sure that users can manage their activities efficiently. It enables users to add tasks by providing a task name, its priority level, and a due date for each task. Tasks will be grouped into categories based on their status, and there will be live counters for pending and finished tasks. There will be also special badges for Urgent and Overdue tasks so that users can pay more attention to them. Users can either mark tasks as finished or delete them when they do not need them anymore, but all tasks will be stored using browser Local Storage so that tasks can be kept after refreshing the page. The application will have an attractive and modern interface inspired by glassmorphism with animated background, responsive design, and controls for a better user experience.

Steps:
Firstly, enter the task name, select its priority (Low, Medium, or High), choose a due date, and click Add.
Then, click Done! to mark a task as completed or Remove to delete it. Urgent and overdue tasks are automatically highlighted.
To track progress you can Monitor the Pending and Completed counters at the top. All tasks are automatically saved in your browser using Local Storage, so they remain available even after refreshing the page.
